MT5 Trader Workbench User Guide

This guide is for first-time users. It focuses on what each core feature does, what to configure first, and where to check when something does not work.

Best for: first launch after installation Recommended order: Settings → Connect → Monitoring Technical analysis reports open in your system browser

1. Quick Start

  1. Open Settings, fill in the MetaTrader 5 Path, and click Verify.
  2. If you run multiple MT5 terminals, enter the full path of the exact terminal you want this app to use.
  3. Go back to Dashboard and click Launch/Reconnect MT5.
  4. After the connection succeeds, you can enable the overlay, create alerts, review order statistics, or generate technical analysis.
For the first run, focus on only two things: verify the MT5 path and connect to MT5.

2. Dashboard

MT5 Status

Shows whether the backend is connected to MT5. Typical states are stopped, running but not logged in, or connected.

Balance / Equity / Profit

Gives you a quick account snapshot.

Margin Level

Helps you judge account risk and margin pressure.
If the dashboard keeps showing Stopped, re-check the MT5 path in Settings first.

3. Price, Volatility, and Indicator Alerts

Price Alerts

  • Enter the symbol, target price, and trigger condition.
  • Common templates only fill the form. Review the example price and click save yourself if you want to create the rule.
  • Use >= for break-above alerts and <= for break-below alerts.
  • The backend validates the rule against the current MT5 price before saving.

Volatility Monitor

  • Use it to detect sharp moves inside a short time window.
  • Lower thresholds and shorter durations make it trigger more easily.

Indicator Alerts

  • Create rules by symbol, timeframe, indicator type, and threshold.
  • Good for automating RSI-style conditions you watch repeatedly.
If Bot push is enabled but no message arrives, go to Settings → Notifications and verify the webhook or token was saved correctly.

4. Order Center

  • Shows today, week, and month profit summaries.
  • Lets you query daily trading statistics by date range.
  • Review metrics are based on daily statistics, so they describe profitable days and average daily/trade figures rather than broker-grade trade-by-trade analytics.
  • Useful for reviewing performance over time instead of only watching current floating P/L.

5. Automation, Sync, and Event Log

Order Broadcast

  • Broadcasts order activity to configured notification channels.
  • It does not create, modify, or close trades.

Order Sync and Local Copy Trading

  • Read the safety notice before enabling either workflow.
  • Confirm account credentials, symbols, contract mappings, and lot multipliers before turning them on.

Event Log

  • Shows currently available local copy trading events, order sync records, and the latest order sync error.
  • It is a lightweight recent-event view, not a complete persisted audit history.
Treat all automation and sync modules as high-risk. Verify the configuration outside the app before relying on any notification or sync result.

6. Technical Analysis

  • Click Generate Technical Analysis to run several analysis modules in sequence.
  • The final combined report opens in your default browser, not inside the desktop app.
  • The current combined flow is mainly designed around XAUUSD.
If report generation fails, first confirm the backend is running and MT5 or other data sources are available.

7. Settings

Connection

  • Interface language: switch between Chinese and English.
  • MT5 path: the most important first-time setup item.
  • Auto-connect: try to connect to MT5 automatically at startup.

Display

  • Overlay symbols, font size, and color: control what the desktop overlay shows.
  • Choose a built-in alert sound or a custom audio file.
  • Use Test Play to preview it immediately.

Notifications

  • Supports DingTalk, WeCom, and Feishu.
  • Enter the config first, click Save Now, then send a test push.

About

  • Configuration migration buttons are disabled until a real import/export API is available.
  • Protect MT5 credentials, TopStep API keys, and bot webhook/token values as sensitive information.

8. FAQ

Q1: The app opens but no data is shown.

  • Check the MT5 status on the Dashboard.
  • Re-verify the MT5 path in Settings.
  • Make sure MT5 is actually logged in, not just opened.

Q2: The price overlay is missing.

  • Click Show Price Overlay on the Dashboard.
  • Check whether the overlay symbol list in Settings is empty.

Q3: Test push notifications fail.

  • Verify the webhook, token, or secret.
  • Make sure the network can reach the target platform.
