MT5 Trader Workbench 使用教程

这份教程面向第一次使用软件的用户，重点说明每个主要功能怎么用、先做什么、出了问题看哪里。

适用场景：首次安装后快速上手 推荐顺序：先设置，再连接，再开监控 技术分析报告会在系统浏览器中打开
快速开始 工作台总览 价格/波动/指标预警 订单中心 自动化/同步 技术分析 系统设置 常见问题

1. 快速开始

  1. 先打开 设置，填写 MetaTrader 5 路径，然后点“验证”。
  2. 如果你只运行一个 MT5 终端，填主程序路径即可；如果有多个终端，请填你要连接的那个完整路径。
  3. 验证通过后，回到 工作台总览，点击“启动/重连 MT5”。
  4. 连接成功后，再根据需要开启价格浮窗、添加预警、查看订单统计或生成技术分析。
建议第一次使用时先只做两件事：验证 MT5 路径 和 启动/重连 MT5。

2. 工作台总览

MT5 状态

查看后端是否连上 MT5。常见状态有：未运行、运行中未登录、已连接。

账户余额 / 净值 / 盈亏

用来快速确认账户当前资金和浮动盈亏。

保证金比例

用于判断账户风险是否偏高，接近危险值时要优先减仓或补充保证金。
如果总览页一直显示“未运行”，优先去设置页重新验证 MT5 路径。

3. 价格 / 波动 / 指标预警

价格预警

  • 填写交易品种、目标价格、触发条件。
  • 常用模板只会填入表单，不会自动创建规则；请先核对示例价格，再手动保存。
  • >= 适合“涨破提醒”，<= 适合“跌破提醒”。
  • 保存前后端会结合当前 MT5 价格做校验。

波动检测

  • 适合监控短时间内的大幅波动。
  • 阈值越小越容易触发，统计时长越短越敏感。

指标预警

  • 可按品种、周期、指标类型、阈值来设置触发规则。
  • 适合把常看的 RSI 条件自动化。
如果你开启了 Bot 推送，但没有收到消息，请去“设置 > 通知”检查 webhook 或 token 是否已经保存成功。

4. 订单中心

  • 可查看当日、本周、本月盈亏概览。
  • 可按开始日期和结束日期查询每日统计明细。
  • 复盘指标基于每日统计，表示盈利日占比和平均日/笔表现，不等同于券商级逐笔交易分析。
  • 适合回看一段时间内的交易表现，而不是只盯着实时浮盈亏。

5. 自动化 / 同步 / 事件日志

订单广播

  • 用于把订单行为广播到已配置的通知通道。
  • 它不会创建、修改或平仓。

订单同步与本地跟单

  • 启用前请先阅读页面中的安全提示。
  • 打开前必须核对账户凭据、品种、合约映射和手数倍数。

事件日志

  • 显示当前可读取的本地跟单事件、订单同步记录和最近一次订单同步错误。
  • 这是轻量近期事件视图，不是完整持久化审计日志。
自动化与同步相关模块都属于高风险功能。依赖任何通知或同步结果前，请先在软件外再次核对配置。

6. 技术分析

  • 点击“生成技术分析”后，软件会串行生成多套分析结果。
  • 最终综合报告会在系统默认浏览器打开，不在应用内预览。
  • 当前综合报告主要围绕 XAUUSD 设计。
如果分析页打开失败，先确认后端服务已启动，再确认 MT5 或数据源是否可用。

7. 系统设置

连接

  • 界面语言：切换中文 / English。
  • MT5 路径：最重要的首次配置项。
  • 自动连接：打开后，软件启动时会自动尝试连接 MT5。

显示

  • 浮窗品种、字体大小、颜色：控制价格浮窗的显示内容。
  • 可选内置提示音，也可选择自定义音频文件。
  • 点击“测试播放”可立即试听。

通知

  • 支持钉钉、企业微信、飞书。
  • 先填配置，再点“立即保存”，最后再点“发送测试推送”。

关于

  • 在真实导入/导出接口可用前，配置迁移按钮会保持禁用。
  • 请把 MT5 凭据、TopStep API Key、Bot webhook/token 当作敏感信息保管。

8. 常见问题

Q1：软件打开了，但没有数据？

  • 先看“工作台总览”里的 MT5 状态。
  • 再去设置页验证 MT5 路径。
  • 确认 MT5 已登录真实或模拟账户，而不是仅启动了空终端。

Q2：价格浮窗没显示？

  • 先在工作台总览点击“开启价格浮窗”。
  • 检查设置里的浮窗品种列表是否为空。

Q3：Bot 测试推送失败？

  • 检查 webhook / token / secret 是否填对。
  • 确认网络可访问对应平台。
你可以随时在应用右上角点击“使用教程”再次打开这份文档。
